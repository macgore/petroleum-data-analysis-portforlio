Petroleum Engineering Data Analytics Portfolio
Python-Based Reservoir, Production & Drilling Engineering Analysis
Overview

This repository documents a structured six-week applied program in Python-driven petroleum engineering analytics.

The objective of this work is to integrate programming, data science, and classical petroleum engineering principles to support:

Production surveillance

Reservoir performance forecasting

Drilling optimization

Economic evaluation

Digital oilfield workflows

All analyses are performed using industry-relevant Python libraries including NumPy, Pandas, Matplotlib, and SciPy, with emphasis on engineering interpretation — not just code execution.

Technical Stack

Programming Language

Python 3

Core Libraries

NumPy (numerical computation)

Pandas (data manipulation & time-series analysis)

Matplotlib (engineering visualization)

SciPy (decline curve regression & curve fitting)

Environment

Jupyter Notebook (cloud-based development via Chromebook)

Repository Structure
Week 1 – Python Foundations for Engineering Computation

Development of core programming fundamentals applied to petroleum engineering calculations:

Unit conversions (pressure, rate, volume)

Conditional operational logic

Loop-based production accumulation

Function-based reusable engineering tools

Week 2 – Production Data Analytics

Application of structured data analysis to well production datasets:

Daily and cumulative production calculations

Gas-Oil Ratio (GOR) evaluation

Water cut monitoring

Data cleaning and validation workflows

Operational threshold flagging

Week 3 – Engineering Visualization & Diagnostics

Professional-grade production trend visualization:

Rate vs. time analysis

Cumulative production curves

Water cut progression

Semi-log decline diagnostics

Multi-well comparison plotting

Focus: translating raw data into actionable engineering insights.

Week 4 – Decline Curve Analysis & Forecasting

Implementation of classical reservoir engineering forecasting methods:

Exponential decline modeling

Hyperbolic decline modeling

Harmonic decline modeling

Nonlinear regression parameter estimation

Estimated Ultimate Recovery (EUR) calculation

Forecast comparison and model sensitivity evaluation

Week 5 – Drilling Performance Evaluation

Quantitative drilling analysis including:

Depth vs. time performance tracking

Rate of Penetration (ROP) calculation

Non-Productive Time (NPT) assessment

Cost per meter analysis

Operational efficiency benchmarking

Week 6 – Integrated Multi-Well Forecast & Economic Screening

Capstone analysis integrating:

Multi-well production forecasting

Decline-based long-term performance modeling

Comparative EUR analysis

Revenue projection modeling

Asset-level performance benchmarking
